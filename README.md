## Getting Started

### Installing

1. ` copy repository`
2. ` npm install`
3. ` npm run build (production)`
4. ` npm start`

# global gulp

```
start: gulp
clear (optional): gulp clear
build production: gulp build

```

## Deployment

If gulp clear don't works, delete dist folder
